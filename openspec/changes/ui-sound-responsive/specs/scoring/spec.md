## MODIFIED Requirements

### Requirement: 채점 결과 표시
시스템은 모든 문제를 완료한 후 맞춘 개수와 총 문제 수를 표시해야 한다(SHALL). 형식: "N / 20". 전체 문제에 대한 풀이 목록도 함께 표시해야 한다(SHALL).

#### Scenario: 채점 결과 화면
- **WHEN** 퀴즈가 완료되고 결과 화면에 진입할 때
- **THEN** 맞춘 문제 수 / 총 문제 수(20)가 표시된다

#### Scenario: 미답 문제 처리
- **WHEN** 답을 입력하지 않은 문제가 있을 때
- **THEN** 미답 문제는 오답으로 처리된다

#### Scenario: 풀이 목록 표시
- **WHEN** 결과 화면에 진입할 때
- **THEN** 20문제 전체가 2열 그리드로 표시되며, 각 항목에 번호, 문제(A ○ B = 정답), 사용자 답이 포함된다

#### Scenario: 정답/오답 시각 구분
- **WHEN** 풀이 목록이 표시될 때
- **THEN** 정답은 초록색, 오답은 빨간색으로 구분되며, 미답은 "미답"으로 표시된다
