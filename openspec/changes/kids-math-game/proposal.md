## Why

아동이 사칙연산(더하기, 빼기, 곱하기, 나누기)을 단계적으로 학습할 수 있는 게임이 필요하다. 난이도를 10단계로 세분화하여 자릿수와 올림/받아내림 여부에 따라 점진적으로 난이도가 올라가도록 구성하고, 점수 이력을 통해 학습 진척도를 확인할 수 있게 한다.

## What Changes

- 사칙연산 선택 화면 추가 (더하기, 빼기, 곱하기, 나누기)
- 난이도 선택 화면 추가 (1~10단계)
- 난이도별 문제 생성 로직 구현 (각 난이도마다 20문제)
  - 난이도 1: 한자리 + 한자리
  - 난이도 2: 두자리 + 한자리 (올림 없음)
  - 난이도 3: 한자리 + 두자리 (올림 없음)
  - 난이도 4: 두자리 + 한자리 (올림 있음)
  - 난이도 5: 한자리 + 두자리 (올림 있음)
  - 난이도 6: 두자리 + 두자리 (일의 자리, 십의 자리 모두 올림 없음)
  - 난이도 7: 두자리 + 두자리 (일의 자리만 올림)
  - 난이도 8: 두자리 + 두자리 (각 자리 올림 있으나 합 100 이하)
  - 난이도 9: 두자리 + 두자리 (합이 100을 약간 초과)
  - 난이도 10: 세자리 + 두자리
- 문제 풀이 화면: 한 문제씩 표시, 이전/다음 네비게이션, 진행 상황 표시
- 답 미입력 시에도 다음 문제로 이동 가능
- 모든 문제 완료 후 채점 결과 화면
- 최근 점수 20개 로컬 저장 (localStorage) 및 점수 이력 조회 화면

## Capabilities

### New Capabilities
- `operation-select`: 사칙연산 종류 선택 및 난이도 선택 화면
- `problem-generator`: 난이도별 문제 생성 로직 (사칙연산 × 10단계)
- `quiz-session`: 문제 풀이 세션 관리 (문제 표시, 이전/다음 네비게이션, 진행률, 답 입력)
- `scoring`: 채점 및 결과 표시, 최근 점수 20개 localStorage 저장 및 이력 조회

### Modified Capabilities
(없음 - 신규 프로젝트)

## Impact

- 프론트엔드: 새로운 웹 앱 페이지들 (선택 화면, 문제 풀이 화면, 결과 화면, 점수 이력 화면)
- 스토리지: 브라우저 localStorage를 사용한 점수 이력 저장
- 외부 의존성: 없음 (순수 웹 기술로 구현)
