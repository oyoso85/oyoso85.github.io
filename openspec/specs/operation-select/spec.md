# operation-select

## Purpose
사용자가 사칙연산(더하기, 빼기, 곱하기, 나누기)과 난이도(1~10)를 선택하여 퀴즈를 시작할 수 있는 화면 흐름을 정의한다.

## Requirements

### Requirement: 사칙연산 선택 화면
시스템은 홈 화면에서 더하기, 빼기, 곱하기, 나누기 4가지 연산을 선택할 수 있는 버튼을 제공해야 한다(SHALL). 점수 이력 조회 메뉴도 함께 표시해야 한다(SHALL).

#### Scenario: 홈 화면 표시
- **WHEN** 사용자가 앱을 열거나 홈으로 돌아올 때
- **THEN** 더하기, 빼기, 곱하기, 나누기 4개 버튼과 점수 이력 버튼이 표시된다

#### Scenario: 연산 선택 시 난이도 화면 이동
- **WHEN** 사용자가 연산 버튼 중 하나를 클릭할 때
- **THEN** 해당 연산의 난이도 선택 화면으로 이동한다

### Requirement: 난이도 선택 화면
시스템은 선택된 연산에 대해 1~10단계 난이도를 선택할 수 있는 화면을 제공해야 한다(SHALL). 각 난이도에 대한 간단한 설명을 표시해야 한다(SHALL).

#### Scenario: 난이도 목록 표시
- **WHEN** 사용자가 연산을 선택한 후 난이도 화면에 진입할 때
- **THEN** 1부터 10까지의 난이도 버튼이 각각의 설명과 함께 표시된다

#### Scenario: 난이도 선택 시 퀴즈 시작
- **WHEN** 사용자가 난이도 버튼을 클릭할 때
- **THEN** 해당 연산과 난이도로 20문제가 생성되고 퀴즈 화면으로 이동한다

#### Scenario: 홈으로 돌아가기
- **WHEN** 사용자가 난이도 화면에서 뒤로가기를 클릭할 때
- **THEN** 홈 화면으로 돌아간다
